# 等高计算 
条件是知道第一列高度比例，让第二列第三列高度等于第一列高度  
公式如下
> `H` : 高度 ； `W` ：宽度 ；  `P` ：高度/宽度 (比例)；  `D/d` ：差值；  `M` : 倍数    
> `H = W1 * P1 = W2 * P2 + D2`    
> 两个宽度关系 `W2 = W1 * M2 + d2`    
> 两个比例的关系 `M2 = P1/P2`     
> 高度相等 `W2 * P2 + D2 = W1 * P1`  
> 带入W1、W2关系  `(W1 * M2 + d2) * P2 + D2 = W1 * P1`  
> 计算后 `D2 = -d2 * P2`  
> 
> 即可设定第二列的宽度 `padding-top: calc(W2 * P2 - d2 * P2)`  
> 这里重点就是计算两个宽度的关系和比例大小
